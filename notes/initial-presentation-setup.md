# Initial Presentation Setup

- The starter deck lives in `main.tex` and uses a minimal `beamer` configuration with the default theme.
- The `compile.sh` script writes PDF output and intermediate LaTeX artifacts to `build/`.
- Example slides are intentionally generic and should be replaced with project-specific material as the research direction sharpens.
- The presentation styling now uses a restrained custom palette: slate text, blue frame headers, teal blocks, and a muted light background.
- If the deck later needs citations, figures, or custom styling, extend the current single-file layout before splitting into multiple TeX inputs.
